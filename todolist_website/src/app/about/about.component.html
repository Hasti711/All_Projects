About Us



“Todoist” is an online todo list making website. In this user
can write their everyday work and save them for future uses
and they can also make checklist to tick mark the work which
has been completed. User can also set timers for some task .
if any user have written a task to complete at 5 pm then
system will notify the user that you have work to complete.

Whenever we have lot many things to do in a day we always ended up
forgetting so many tasks.using our website your work become very easy
you just have to write down all the task you are supposed to do in a day
and you are good to go. This way you can easily complete your work
without forgetting anything and also you can cancel out the task which
are completed.



One of the most important reasons you should use
a to do list is that it will help you stay organised.
When you write all your tasks in a list, they seem
more manageable. When you've got a clear outline
of the tasks you've got to do and those you've
completed, it helps you stay focused.
